## ✅ 기능 목록

- [ ] 음식점 클릭시 해당 음식점 정보가 담긴 모달창을 보여준다.
- [ ] 모달창 닫기 버튼 클릭시 모달창이 닫힌다.
- [ ] 카테고리 선택시 해당 카테고리만 담은 음식점 목록을 보여준다.
- [ ] 이름순 선택시 가나다순으로 정렬한 음식점 목록을 보여준다.
- [ ] 거리순 선택시 거리가 가까운 순서로 정렬한 음식점 목록을 보여준다.
